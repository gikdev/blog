What is a website?

Before diving into details of a websites and making them, it’s really good to consider that what a website is? A website is actually some files which are uploaded to a 24/7 running computer called server.

The files are actually different based on the types of the websites.

Types of websites

There are 2 types of websites: static and dynamic.

Static websites have prebuilt files that are made with HTML, CSS & JavaScript languages. They can be also generated via WYSIWYG editors or SSG. When a user wants to see a static website, servers give browsers the files without manipulating them.

On the other hand, dynamic websites have programs which build the pages when a user wants to see a website. They also return HTML, CSS, and JavaScript files but they can manipulate them before giving them to you. Dynamic servers mostly use programming languages to make that programs, e.g. PHP, ASP.NET, Python etc.

When the files are given to the browser, the browser renders them with it’s engine and shows it to you!

Static WebsiteDynamic Website

How they work?

As I said, the files are given to browsers and browsers take care of rendering and showing them to you.

Making your first webpage

So are you ready to make your first webpage? So, let’s begin.

First, start by making a new file called site.html on your PC. You can achieve this, by simply right-clicking on desktop and choosing New > Text Document. Then you just name the file.

Now open the file with Notepad. Right-click on the file: Open With > Notepad. Now start writing anything you want in it! For example write: Hello. I’m making a webpage. After that make sure you save it with Ctrl+S.

Then open chrome or edge if you have. Drag & drop the file into browser window to open the file.

Congrats! You just made a very simple webpage. But you’ve seen that in a website we have headings, buttons, forms, links, etc. For that you need to learn HTML. I may give you a list of courses to take.

But I’m gonna tell you the nitty-gritties. So let’s get started.

HTML; The very basics

Every HTML file consists of several elements. Elements are made with combining tags and content. For example:

<p> Hello World </p>

Here, <p> and </p> are tags. And Hello World is the content. But the whole thing is an element.

So if you wanted to know how to make for instance, a heading, you should know the tag name. Here is a very basic list of HTML tags.

  • h1: Level 1 heading
  • h2: Level 2 heading
  • h3: Level 3 heading
  • h4: Level 4 heading
  • h5: Level 5 heading
  • h6: Level 6 heading
  • button: A clickable button
  • p: A paragraph
  • strong: Make the text bold

So here is a very simple HTML code block and it’s look in browser.

<h1>What I am learning</h1>
<p>I'm learning how to make <strong>webpages</strong></p>
<button>Click me!</button>
