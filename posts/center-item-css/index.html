How to center an item in CSS

So if you’ve started learning CSS or you’re doing projects with it, you’ve come across the situation where you want an element to be in the center of it’s parent. You might think that it’s hard but it’s so easy and there are lots of ways.

There are various methods for getting the job done, ranging from absolute positioning, all the way to flexbox. We’ll start with the easiest…

Center Items with CSS Grid:

For this method all you need to do is to make the parent have the grid display. So:

.parent {
  display: grid;
}

Then you have to place items in center:

.parent {
  display: grid;
  place-items: center;
}

Center Items with CSS FlexBox:

This one is also simple but you might need basic knowledge of CSS Flexbox in order to not get confused with it. This is the code:

.parent {
  display: flex;
  justify-content: center;
  align-items: center;
}

Let me explain:

  • display: flex; makes the container so that it’s a flex container
  • justify-content: center; aligns the children of this container in the main-axis which in this example is horizonal
  • align-items: center; aligns the children of this container in the cross-axis which in this example is vertical

Center Items with Absolute Positioning:

This is the method that I don’t recommend. First apply a relative position to the parent:

.parent {
  position: relative;
}

Then you should center the child with top and left properties:

.child {
  position: absolute;
  top: 50%;
  left: 50%;
}

But, even after these there would be a little problem. The thing that has been centered is the top left point of the element. So you have to move the thing to the center like this:

.child {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
}

Center text:

So if there is a simple text you can just do this:

.parent {
  text-align: center;
  vertical-align: middle;
}

Conclusion:

So you learned 4 ways to center things with CSS. Hope you enjoyed it & have a great week ahead.
